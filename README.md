# Thorium keyboard
![](./img/thorium.jpg)
## Features
- ortholinear  
- 34-keys
- wireless
- split
- dongle  
- MX switches, hotswap 
- small PCB (96mm*77mm)
- 3D printed cases and plates
- Lipo battery (260mAh)
- Seeed Studio XIAO nRF52840
- ZMK Firmware
